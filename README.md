# 后端日常 Agent 提示词库

本仓库是一个为后端开发者准备的 **Agent 提示词（Prompt）集合**，用于日常开发、排查、优化、测试和代码重构等场景。

## 📚 目录结构

- `all.md` - 通用提示词集合，适用于多种后端语言和场景。
- `common.md` - 常见提示词与模板，可直接复用。
- `Golang/` - Go 语言相关提示词（Gin、GoZero 等）。
- `Java/` - Java 语言相关提示词（Spring 等）。
- `Node/` - Node.js 相关提示词（Express、NestJS 等）。
- `Python/` - Python 相关提示词（Django、FastAPI、Flask 等）。

## 🎯 使用说明

1. 打开你关心的语言/框架文件（例如 `Python/FastAPI.md`）。
2. 复制你需要的 prompt 片段。
3. 在 Agent（如 ChatGPT、Copilot、LangChain 等）中使用，以获得更一致、更专业的后端帮助。

## ✨ 适用场景

- 需求拆解、设计评审、架构讨论。
- 代码实现、Bug 定位、性能优化。
- 测试用例生成、错误处理、日志追踪。
- 文档撰写、代码注释、规范检查。
